data structure:
the folder "source" contains originally monthly data files from luftdaten.info(not uploaded, about 6GB per file)

the folder "raw" contains filtered source data that contain measurements from a specific region. (not uploaded, 200 MB per file)

the folder "mock" contains sample data for the prototype (depricated - not used at this moment)

the folder "sensors" contains data about sensors for a specific city at a specific time

the folder "avg" contains the prepared data that is plotted in the diagrams

the folder "year" contains data of monthly files that are merged together in data prepreparation to year data.
